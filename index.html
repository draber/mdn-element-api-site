The totally unofficial MDN Element API

A while ago, I worked on a project where I needed a reference of all HTML, SVG and MathML elements, their attributes, and properties. Preferably well-structured and in JSON format. In fact, something like an MDN API, which is something that doesn't exist - or at least didn't until now. I found tons of kinda-similar resources on the web, but none were really to my liking.

And anyway, don't we all love to re-invent the wheel because we secretly know that our version will be the roundest and wheeliest wheel ever known to humankind? You see where this is going. I made one.

On a side note, the project I originally needed the API for is now lingering in the limbo of projects, that seemed to be a great idea at the time. It now has way too many branches and I can hardly remember the original train of thoughts.

The list of elements

You can access the data in two different formats, either as one big JSON resource with everything in it or on a per-element basis.

The links to the per-element resources are listed below. This part is generated from said one big JSON resource, which is pretty meta.
